Install PureGuard

Follow the steps below for your browser

Chrome
Edge
Brave
Firefox
1

Open Extensions Page

Type this in your address bar and press Enter:

chrome://extensions
2

Enable Developer Mode

Find the "Developer mode" toggle in the top-right corner and turn it ON.

3

Load the Extension

Click "Load unpacked" button that appears, then select THIS folder (the one containing this file).

4

Pin the Extension

Click the puzzle piece icon in Chrome's toolbar and pin PureGuard for easy access.

✓ You're Protected!

PureGuard is now active and protecting your browsing.
